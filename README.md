## 기능 구현 목록
- 회원 기능
```
- 회원 가입 기능
- 로그인 기능
- 회원 목록 조회 기능 (이름, 학교만 조회 가능)
- 회원 정보 수정 기능 (회원 목록 하이퍼링크를 통해 수정 폼으로 이동)
```

- 글 쓰기 기능
```
- 글 목록 조회 기능 (전체 글 조회 가능)
- 글 수정 기능 (동일 회원이 작성한 글만 수정 가능)
- 글 삭제 기능 (동일 회원이 작성한 글만 삭제 가능)
```
